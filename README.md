# OpenCode Tools

This repository contains the source code and tooling for internal tools used by the OpenCode AI agent.

## Project Structure

- `tool-cli`: Internal CLI tools used by the agent.
- `tool-lib`: Common library for agent tools.

## Overview

These tools provide the necessary capabilities for the OpenCode AI agent to interact with the system, execute commands, and perform software engineering tasks efficiently.

For the main OpenCode project, please visit [opencode.ai](https://opencode.ai).
